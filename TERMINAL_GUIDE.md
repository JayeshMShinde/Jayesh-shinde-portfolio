# Terminal Portfolio Guide

## Overview
Your portfolio now features an interactive terminal interface that provides command-line access to all your professional information while preserving the original visual design.

## How to Access
- **Visual Mode**: Your existing beautiful portfolio interface
- **Terminal Mode**: Click the terminal icon (bottom-right) to open the command-line interface
- **Hybrid Experience**: Switch seamlessly between both modes

## Terminal Features

### Core Commands
```bash
help            # Show all available commands
whoami          # Personal information with ASCII art
about           # Detailed professional background
skills          # Technical skills organized by category
experience      # Work history with responsibilities
projects        # Featured projects with tech stacks
education       # Educational background
contact         # Contact information and availability
social          # Social media and professional profiles
resume          # Resume download options
```

### Navigation Commands
```bash
clear           # Clear terminal screen
gui             # Switch to visual portfolio mode
exit            # Close terminal
banner          # Show ASCII logo
```

### Interactive Features
- **Auto-completion**: Type partial commands and press Tab
- **Command history**: Use ↑/↓ arrows to navigate previous commands
- **Suggestions**: Real-time command suggestions as you type
- **Easter eggs**: Try commands like 'coffee' or 'sudo'

## Usage Tips

### Getting Started
1. Click the terminal icon to open
2. Type `help` to see all commands
3. Use `whoami` for a quick introduction
4. Try `banner` for the ASCII logo
5. Use `gui` to return to visual mode

### Power User Features
- Tab completion for faster navigation

### Best Practices
